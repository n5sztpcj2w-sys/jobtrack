# JobTrack

> **[BETA] Version 1.0.0** - This is a beta release and may include bugs or rough edges. Use with that in mind.
>
> JobTrack is a tool for keeping track of your job applications in one place. Instead of juggling spreadsheets or trying to remember where you applied, you can log each application, update its status as things progress, and keep notes on interviews and companies - all in one spot.
>
> ---
>
> ## Features
>
> - Log and manage job applications
> - - Track the status of each application through the hiring process
>   - - Keep track of interview dates and schedules
>     - - Store notes and details about companies you have applied to
>      
>       - ## Installation
>      
>       - ```bash
>         git clone https://github.com/n5sztpcj2w-sys/jobtrack.git
>         cd jobtrack
>         ```
