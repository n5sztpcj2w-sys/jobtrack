# JobTrack 🚀

> ⚠️ **ベータ版 (Beta v0.1.0)** - 現在ベータテスト中です。バグや不具合が含まれる場合があります。
>
> 求人・就活の応募状況を一元管理するツールです。
>
> ## 機能 (Features)
>
> - 求人応募の記録・管理
> - - 選考ステータスのトラッキング
>   - - 面接日程の管理
>     - - 応募企業の情報整理
>      
>       - ## インストール (Installation)
>      
>       - ```bash
>         git clone https://github.com/n5sztpcj2w-sys/jobtrack.git
>         cd jobtrack
>         ```
